# Becky's Cryptid Adventure

A cryptid-hunting incremental game. Whisper campfire legends, sail past the edge of the map, snap blurry photos of Bigfoot, and finally — uncover what's really out there.

Built on the same Profectus-based engine as Playbook. Four eras of escalating weirdness:

1. **Whispers in the Woods** — primordial folklore (wendigos, fae, dragons)
2. **Beyond the Map** — Age of Sail sea monsters (kraken, mermaids, sirens)
3. **Caught on Film** — modern sightings era (Bigfoot, Nessie, Yeti, Mothman)
4. **They Are Among Us** — UFOs, abductions, Men in Black, disclosure
