Button Language Mockups

Ten candidate button systems for Playbook 2. Each row shows the same set of actions (BUY · BUY disabled · HIRE manager · UPGRADE · bulk-multiplier picker) so you can compare them side-by-side. Pick the language that reads cleanest to you and I'll apply it across the app — tile actions, prestige, settings, everything. Theme palette here is Era 2 (Printing Press).
